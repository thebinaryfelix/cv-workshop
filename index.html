2 HOUR CURRICULUM

Intro to HTML and CSS by Ironhack

1. SETUP

  1. Start Visual Studio Code
  2. Create a folder named CV on your desktop.
  3. Drag and drop this folder inside VS Code window.
  4. Now inside VS Code: On the left window
    • Right click and then click on New File
    • Name it index.html and Enter
    • Repeat the process and save the new file as style.css
    • Right click and then New Folder
    • Name it images and Enter
  5. Open your favorite browser (Chrome, Safari, Firefox...)
  6. Drag your index.html file into the browser
  7. We're all set!

2. CONTENT

  1. Let's make a curriculum for Homer Simpson
  2. To use this image, click on it and after downloading, move it to your images folder.
  3. Next, this will be our starting code inside the index.html file:

3. Bootstrap - Font Awesome

  1. Copy the following lines
    2. <meta http-equiv="X-UA-Compatible" content="IE=edge"> <meta name="viewport" content="width=device-width, initial-scale=1"> <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.1.2/css/bootstrap.min.css" integrity="sha384-Smlep5jCw/wG7hdkwQ/Z5nLIefveQRIY9nfy6xoR1uRYBtpZgI6339F5dgvm/e9B" crossorigin="anonymous"> <link rel="stylesheet" href="https://use.fontawesome.com/releases/v5.1.0/css/all.css" integrity="sha384-lKuwvrZot6UHsBSfcMvOkWwlCMgc0TaWr+30HWe3a4ltaBwTZhyTEggF5tJv8tbt" crossorigin="anonymous"> <link rel="stylesheet" href="style.css">
  2. Paste the code you just copied inside the head tag in your index.html file.
  3. Should be something like this:
    4. <head> <meta charset="UTF-8"> <meta http-equiv="X-UA-Compatible" content="IE=edge"> <meta name="viewport" content="width=device-width, initial-scale=1"> <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.1.2/css/bootstrap.min.css" integrity="sha384-Smlep5jCw/wG7hdkwQ/Z5nLIefveQRIY9nfy6xoR1uRYBtpZgI6339F5dgvm/e9B" crossorigin="anonymous"> <link rel="stylesheet" href="https://use.fontawesome.com/releases/v5.1.0/css/all.css" integrity="sha384-lKuwvrZot6UHsBSfcMvOkWwlCMgc0TaWr+30HWe3a4ltaBwTZhyTEggF5tJv8tbt" crossorigin="anonymous"> <link rel="stylesheet" href="style.css"> <title>Homer J Simpson</title> </head>

4. Wrap the blocks

  1. Wrap each block with a div tag
    2. <div> <div> <!-- Block #1 --> <h1>Homer Jay Simpson</h1> <p>Versitile employee, currently working at Springfield's Power Plant</p> <p>This can be a longer description about the person's goals</p> </div> <div> <!-- Block #2 --> <img src="images/profile.jpg" alt=""> </div> <div> <!-- Block #3 --> <div> <ul> [ ... ] </ul> </div> <div> <ul> [ ... ] </ul> </div> </div> </div> <div> <!-- Block #4 --> <h2>EDUCATION</h2> <h3>High School Degree</h3> <h4>Springfield High</h4> <p>Start Date - End Date</p> <p>Springfield, USA</p> <p>Course</p> <ul> <li>Completed High School while eating lots of donuts.</li> </ul> </div> <div> <!-- Block #5 --> <h2>SKILLS</h2> <ul> <li>Math</li> <li>English</li> <li>Art</li> <li>Biology</li> <li>Chemestry</li> <li>Gardening</li> </ul> </div>
  2. Attention! On block#3 put each ul inside a div
  3. Now, blocks #1, #2 and #3 all go inside another div

5. Naming tags - Bootstrap classes

Bootstrap has special classes for the Grid system. We'll use some of them now.

  • For block#1, apply the following classes to the div tag: block-1 col-lg-8
  • For the first p on block#1, apply the tag high-text
  • For block#2, apply the following classes to the div tag: block-2 col-md-4
  • For block#3, apply the following classes to the div tag: block-3 col-md-8
  • For block#4, apply the following classes to the div tag: block-4 col-lg-6
  • For block#5, apply the following classes to the div tag: block-5 col-lg-6

Next, let's wrap some more...

  1. Wrap blocks #1, #2, #3 inside a div and apply these classes to it: row row-1
  2. Wrap blocks #4 and #5 inside a div and apply the following classes: row m-0 mt-4
  3. Finally, wrap all the code inside the body tag with a div and apply the class container

6. The CSS

Inside your style.css file, copy and paste this code:

@import url('https://fonts.googleapis.com/css?family=Roboto:400,500,500i,900'); body{ font-family: 'Roboto-Mono', monospace; padding: 0; margin: 0; font-size: 16px; -webkit-print-color-adjust:exact; } h1{ font-size: 2.5em; font-weight: bold; } h2{ font-size: 2.0em; color: #EC3A35; } h3{ font-size: 1.6em; } h4{ font-size: 1.3em; } .high-text{ color: #2DC5FA; font-weight: bold; } .row-1{ background-color: #181B23; } .block-1{ color: #ffffff; padding: 15px; } .block-2{ text-align: center; padding: 15px; } .block-2 img{ width: 75%; border: 6px solid #2DC5FA; border-radius: 50%; padding: 10px; background-color: #ffffff; } .block-3 div{ display: inline-block; padding: 0; } .block-3 li{ list-style-type: none; line-height: 2.0em; } .block-3 i{ margin-right: 10px; } .block-3 a{ color: #ffffff; text-decoration: none; } .block-5 li{ list-style-type: none; display: inline; line-height: 3.0em; border: 2px solid #EC3A35; border-radius: 5px; padding: 5px 10px 5px 10px; } .block-5 li:hover{ background-color: #181B23; color: #ffffff; border: 2px solid #2DC5FA; }

7. Deploy!

It's time to put the CV online!

  1. Create a folder in Google Drive and share it
  2. Select the option "Anyone on the internet can find and view"
  3. Access DriveToWeb and choose the Google Drive option
  4. Login with your account and allow it to access your files (if you feel unconfortable with that, create a Gmail account just to store your public files)
  5. It's online! On the Admin Panel select the link that corresponds to your CV and that's it :)
