2 HOUR CURRICULUM

Intro to HTML and CSS by Ironhack

1. SETUP

  1. Start Visual Studio Code
  2. Create a folder named CV on your desktop. Drag and drop this folder inside VS Code window.
  3. Now inside VS Code: On the left window
    • Right click and then click on New File
    • Name it index.html and Enter
    • Repeat the process and save the new file as style.css
    • Right click and then New Folder
    • Name it images and Enter
  4. Open your favorite browser (Chrome, Safari, Firefox...)
  5. Drag your index.html file into the browser
  6. We're all set!
