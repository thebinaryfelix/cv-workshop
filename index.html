2 HOUR CURRICULUM

Intro to HTML and CSS by Ironhack

1. SETUP

  1. Start Visual Studio Code
  2. Create a folder named CV on your desktop.
  3. Drag and drop this folder inside VS Code window.
  4. Now inside VS Code: On the left window
    • Right click and then click on New File
    • Name it index.html and Enter
    • Repeat the process and save the new file as style.css
    • Right click and then New Folder
    • Name it images and Enter
  5. Open your favorite browser (Chrome, Safari, Firefox...)
  6. Drag your index.html file into the browser
  7. We're all set!

2. CONTENT

  1. Let's make a curriculum for Homer Simpson
  2. To use this image, click on it and after downloading, move it to your images folder.
  3. Next, this will be our starting code inside the index.html file:
    4. <!DOCTYPE html> <html> <head> <meta charset="UTF-8"> <title>Homer J Simpson</title> </head> <body> <h1>Homer Jay Simpson</h1> <p>Versitile employee, currently working at Springfield's Power Plant</p> <p>This can be a longer description about the person's goals</p> <img src="images/profile.jpg" alt=""> <ul> <li>homer.simpson@gmail.com</li> <li>Springfield, OR - USA</li> <li>@homersimpson_official</li> </ul> <ul> <li>+1 503-655-2393</li> <li>May 12, 1956</li> <li>linkedin.com/in/homersimpson</li> </ul> <h2>EDUCATION</h2> <h3>High School Degree</h3> <h3>Springfield High</h3> <p>Start Date - End Date</p> <p>Springfield, USA</p> <p>Course</p> <ul> <li>Completed High School while eating lots of donuts.</li> </ul> <h2>SKILLS</h2> <ul> <li>Math</li> <li>English</li> <li>Art</li> <li>Biology</li> <li>Chemestry</li> <li>Gardening</li> </ul> </body> </html>
  4. For every alteration on your index.html or style.css file, you have to save it by pressing CTRL + S and Reload the page with F5 or CTRL + R to see the result.

3. CSS - Bootstrap

  1. Copy the following line (click 3 times fast on any word to select all the text)
    2. <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.1.2/css/bootstrap.min.css" integrity="sha384-Smlep5jCw/wG7hdkwQ/Z5nLIefveQRIY9nfy6xoR1uRYBtpZgI6339F5dgvm/e9B" crossorigin="anonymous">
  2. Paste the code you just copied inside the head tag in your index.html file.
  3. Should be something like this:
    4. <head> <meta charset="UTF-8"> <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.1.2/css/bootstrap.min.css" integrity="sha384-Smlep5jCw/wG7hdkwQ/Z5nLIefveQRIY9nfy6xoR1uRYBtpZgI6339F5dgvm/e9B" crossorigin="anonymous"> <title>Homer J Simpson</title> </head>
