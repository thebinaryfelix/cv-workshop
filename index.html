2 HOUR CURRICULUM

Intro to HTML and CSS by Ironhack

1. SETUP

  1. Start Visual Studio Code
  2. Create a folder named CV on your desktop.
  3. Drag and drop this folder inside VS Code window.
  4. Now inside VS Code: On the left window
    • Right click and then click on New File
    • Name it index.html and Enter
    • Repeat the process and save the new file as style.css
    • Right click and then New Folder
    • Name it images and Enter
  5. Open your favorite browser (Chrome, Safari, Firefox...)
  6. Drag your index.html file into the browser
  7. We're all set!

3. CSS - Bootstrap - Font Awesome

  1. Copy the following lines
    2. <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.1.2/css/bootstrap.min.css" integrity="sha384-Smlep5jCw/wG7hdkwQ/Z5nLIefveQRIY9nfy6xoR1uRYBtpZgI6339F5dgvm/e9B" crossorigin="anonymous"> <link rel="stylesheet" href="https://use.fontawesome.com/releases/v5.1.0/css/all.css" integrity="sha384-lKuwvrZot6UHsBSfcMvOkWwlCMgc0TaWr+30HWe3a4ltaBwTZhyTEggF5tJv8tbt" crossorigin="anonymous"> <link rel="stylesheet" href="style.css">
  2. Paste the code you just copied inside the head tag in your index.html file.
  3. Should be something like this:
    4. <head> <meta charset="UTF-8"> <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.1.2/css/bootstrap.min.css" integrity="sha384-Smlep5jCw/wG7hdkwQ/Z5nLIefveQRIY9nfy6xoR1uRYBtpZgI6339F5dgvm/e9B" crossorigin="anonymous"> <link rel="stylesheet" href="https://use.fontawesome.com/releases/v5.1.0/css/all.css" integrity="sha384-lKuwvrZot6UHsBSfcMvOkWwlCMgc0TaWr+30HWe3a4ltaBwTZhyTEggF5tJv8tbt" crossorigin="anonymous"> <link rel="stylesheet" href="style.css"> <title>Homer J Simpson</title> </head>

5. Naming tags - Bootstrap classes

Bootstrap has special classes for the Grid system. We'll use some of them now.

  • For block#1, apply the following classes to the div tag: block-1 col-lg-8
  • For the first p on block#1, apply the tag high-text
  • For block#2, apply the following classes to the div tag: block-2 col-lg-4
  • For block#3, apply the following classes to the div tag: block-3 col-lg-8
